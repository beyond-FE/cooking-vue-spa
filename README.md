# wechat-mall
> A vue project.

## Development

```shell
npm i cooking-cli -g
npm run dev
cooking watch -p
```

## Production
```
npm run dist

cooking build -p
```

## 开发目录
```

src //资源文件

src-pages  //放置页面组件

compoents // 公用组件

api // 请求路径

routes //路由

store //全局状态

assets // 静态资源文件

static   //图片、字体文件
```
